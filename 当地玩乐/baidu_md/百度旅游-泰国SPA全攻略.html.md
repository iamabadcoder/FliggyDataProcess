# 按摩种类
### 泰国按摩项目很多，有面部护理、精油按摩、身体磨砂、泰式按摩、脚底精油按摩和手部精油按摩等等。根据护理的部位不同，时间也从半个小时到三个小时不等。一般做一个足部按摩的价格是120泰铢到200泰铢，做一个全身精油按摩是250泰铢到450泰铢。卸下外衣躺在按摩的床上，听着轻音乐，享受这全身心的放松。按完之后，你就会感到神清气爽，气定神闲。下面重点介绍泰国最热门的几项按摩套餐。
## 1. 泰式（Traditional Thai Massage）：
### 泰式按摩为泰国古代医学文化之一，拥有四千多年历史，源远流长。古代泰国皇族利用它作为强身健体和治疗身体劳损方法之一。近年来经过泰国政府监管发展和积极推广的情况下，泰式按摩已广为人知，成为受中外重视的天然治疗。
## 2. 精油按摩（Oil Massage）：
### 精油按摩是指用精油按摩油（基础油和精油调配好的按摩油），涂抹在需要的部位，进行按摩的方法主要作用可以使你全身放松，适合奔波了一天的游客们。利用精油来按摩，可以达到放松心情，纾解压力的功效，而同时有利于让肌肤解压，也是养生美容的一大妙法。
## 3. 足部按摩Reflexology（Foot Massage）：
### 足部按摩是众多按摩术里面的一个分支，由于足部神经分布密集，又分别有身体各个部位有着密切的关系，所以足部按摩逐渐得被得到重视，旨在恢复身体里能量的平衡，主要按脚的穴位，但也会按手指和耳朵。
## 4. 瑞典按摩（Swedish Massage）：
### 瑞典按摩，传统的欧洲按摩技术，是最容易接受的一种按摩方式。利用精油按摩肌肉，增强体液循环，为身体进行大扫除。按摩通常从脚心或手心开始，沿着血液流向心脏的方向推动，主要以抚推、轻扫、揉搓、拍捶几组动作，以中度压力按摩。
